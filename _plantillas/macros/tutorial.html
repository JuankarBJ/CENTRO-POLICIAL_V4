Guía de Funciones

Asistente de Atestados y Gestión de Plantillas

Abrir centro de gestión y descarga

Inicia el portal web de plantillas. Esta es la herramienta principal para encontrar y preparar tus documentos.

Cuándo usarlo: Úsalo para buscar en la base de datos de plantillas, descargarlas a tu "carrito" de trabajo, y **prepararlas** para su rellenado con los datos que generes.

Realizar autocompletado seleccionando manualmente datos y plantillas

Esta es la función de rellenado más flexible. Te permite rellenar cualquier plantilla de Word con datos de cualquier archivo CSV.

Cuándo usarlo: Ideal cuando los archivos .csv y .docx NO se llaman igual, o cuando quieres usar un mismo CSV (ej. implicado.csv) para rellenar varias plantillas (ej. acta.docx, derechos.docx, etc.).

Proceso:

  1. Seleccionas los archivos CSV que contienen tus datos (puedes elegir varios).
  2. Seleccionas las plantillas DOCX que quieres rellenar (puedes elegir varias).
  3. Seleccionas la carpeta donde se guardarán los documentos rellenados.

Generar csv con datos fusionados de otros csv

Esta función combina varios archivos CSV en un único archivo nuevo.

Cuándo usarlo: Cuando tienes datos repartidos en varios CSV (ej. implicadoA.csv, implicadoB.csv, vehiculo.csv) y quieres crear un solo archivo maestro (Salida_Combinada.csv) que contenga todos los campos y valores juntos.

Proceso:

  1. Seleccionas todos los CSV que quieres combinar.
  2. La macro los lee todos y crea un nuevo archivo Salida_Combinada.csv en la carpeta de este Excel.

Añadir datos de csv fuente en otros csv

Esta función "inyecta" los datos de un CSV "Fuente" dentro de otros CSV "Destino".

Cuándo usarlo: Es el caso de uso más común. Tienes un CSV con los datos del implicado (implicado.csv) y quieres que esos datos se añadan a los archivos de datos de cada acta (alcoholemia.csv, derechos.csv, etc.).

¡Importante! Esta acción SOBREESCRIBE los archivos CSV "Destino" (alcoholemia.csv, derechos.csv) con la nueva información combinada.

Proceso:

  1. Seleccionas el CSV "Fuente" (ej. implicado.csv).
  2. Seleccionas los CSV "Destino" (ej. alcoholemia.csv, derechos.csv).
  3. Los archivos destino son actualizados al instante.

Autocompletar automáticamente en carpeta

Esta es la función de rellenado automático principal. Escanea una carpeta y rellena plantillas basándose en la coincidencia de nombres.

Cuándo usarlo: Cuando tienes una carpeta de trabajo bien organizada donde cada plantilla .docx tiene su archivo .csv con el mismo nombre exacto.

Ejemplo:

  • Acta de Derechos.docx se rellenará con Acta de Derechos.csv
  • Inspección Vehículo.docx se rellenará con Inspección Vehículo.csv

Proceso:

  1. Seleccionas la carpeta principal del atestado.
  2. La macro busca todos los pares .docx/.csv que coincidan.
  3. Guarda todos los documentos rellenados en una nueva subcarpeta llamada documentos generados.
